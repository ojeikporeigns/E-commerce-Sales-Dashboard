# Sales Performance Analysis Dashboard (2024–2025)

# Project Overview

This project is a Sales Performance Analysis built using both:

Microsoft Excel

Microsoft Power BI


The objective is to transform raw sales data (2024–2025) into meaningful business insights through structured analysis, KPI tracking, and interactive dashboards.

This project demonstrates strong skills in data analysis, business intelligence, and dashboard development.


---

🗂 Project Files

📁 Sales_Data_Excel.xlsx – Full Excel analysis workbook

📁 Sales_Dashboard.pbix – Interactive Power BI dashboard

Power BI

3. Explore dynamic visualizations and KPI cards.
